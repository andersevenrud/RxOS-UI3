# OS.js Skylark

Conversion of https://github.com/Othernet-Project/RxOS-UI2 into OS.js v3

## Requirements

Node 8 or above

## Installation

```
# Install dependencies
npm install

# Discover packages
npm run package:discover

# Build client
npm run build

# Build packages
./build.sh
```

## Running

`npm run serve`

Launches at http://localhost:8080 by default
